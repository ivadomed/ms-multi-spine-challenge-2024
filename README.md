# The MS-Multi-Spine Miccai 2025 Challenge

Deep learning models for the MS-Multi-Spine challenge 2024: https://portal.fli-iam.irisa.fr/ms-multi-spine/

More information can be found in the Zenodo record: https://zenodo.org/records/14051168

## Submissions

In this challenge, two submissions, were done: 
- the first strategy consisted of a monomodal model: code can be accessed in this folder [monomodal-model](monomodal-model).
- the second strategy consisted of a multimodal model: code can be accessed in this folder [multimodal-model](multimodal-model).

Models for both versions are available in the latest release.
