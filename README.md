# ms-spine-challenge
Deep learning models for the MS-Multi-Spine challenge 2024: https://portal.fli-iam.irisa.fr/ms-multi-spine/

More information can be found in the Zenodo record: https://zenodo.org/records/14051168


# Unregistered Approach 

The goal of this branch is to try to developp and unregistered approach for the ms challenge. 

The idea is the following: 
- encode the inputs using SAM 
- use a ROI approach to fuse the information 
- output the segmentation on the T2w image
